## Decision Records

Major technical decisions must be recorded in `/docs/decisions`.

Use the format:

`ADR-###-short-decision-name.md`

Examples:

* `ADR-001-use-flutter-for-mobile.md`
* `ADR-002-use-nestjs-for-core-api.md`
* `ADR-003-offline-outbox-synchronization.md`
* `ADR-004-lesson-content-versioning.md`
