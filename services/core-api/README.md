# Prolific Core API

Sprint 0 NestJS scaffold for the Prolific versioned REST API.

The application currently contains only the Nest bootstrap and an empty root module. Product features, authentication, database entities, lesson APIs, and synchronization behaviour are intentionally not implemented.

## Commands

Run these from the repository root:

```text
npm run dev:api
npm run lint:api
npm run test:api
npm run test:api:e2e
npm run build:api
```

Public API routes will use the `/api/v1` prefix when feature modules are introduced.
