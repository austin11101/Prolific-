# Prolific Mobile

Sprint 0 Flutter scaffold for the Prolific learner mobile application.

Only Android and iOS platform projects are generated. The current application is Flutter's empty template; onboarding, lesson browsing, offline storage, synchronization, and reading-player behaviour are intentionally not implemented.

## Commands

Run these from the repository root:

```text
npm run dev:mobile
npm run analyze:mobile
npm run test:mobile
npm run format:mobile
```
